About the engine
Data sources by sport:
• MLB: statsapi.mlb.com (official, no key) — batter game logs, pitcher ERA/WHIP, probable starters
• MLB weather: open-meteo.com (no key) — wind, temp, precipitation at stadium
• NHL: api-web.nhle.com (official, no key) — player game logs, goalie SV%/GAA, TOI
• NBA/WNBA: BallDontLie (optional key) → ESPN fallback
• NFL: ESPN game logs (snap/target share unavailable free)
• Sportsbook lines: The Odds API (free key from the-odds-api.com)

Model uses EWMA mean + normal distribution to compute P(Over) and P(Under), then compares against implied probability from sportsbook odds to find +EV edges. Minimum 5 valid games required. No randomness. All data stored locally on your device.
