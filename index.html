About the engine
PropEdge fetches live game logs, injury reports, and schedule data directly from ESPN's public API and MLB's official API. Projections are built using exponential-decay weighted performance (L5/L10), matchup-adjusted opponent defensive rankings, pace/environment factors, and regression-to-mean modeling. Sportsbook lines come from The Odds API (requires free key). All data and history lives in your browser's localStorage — nothing is sent to any server.
